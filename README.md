# Take-Note

## Description
Take Note is an application designed for note-taking, organizing, and task management.

## Installation
* Download all starter files from repo
* Run npm install 
* Run node server.js
* Deploy with Heroku

## Technologies
* Express 
* Heroku 
* Node
* Express
* npm
